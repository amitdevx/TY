# SPPU TYBSc Computer Science

This repository is created to help TYBSc Computer Science students of Savitribai Phule Pune University (SPPU) across multiple semesters. Students can share practicals, assignments, and help each other with semester work.

## What You'll Find Here

Practical programs, assignment solutions, lab work, and reference materials for Semester 5 & 6 - shared only for learning and understanding concepts.

## Repository Structure

- **01-Semester-V** – Core Java, Operating Systems, Web Technology-I, Theory of Computer Science, Major Electives (Data Science / Database / Embedded Systems), and VSC (AI & Machine Learning).
- **02-Semester-VI** – Advanced Java, Design Framework, Web Technology-II, Compiler Construction, Major Electives (Android / Software Testing / IoT), and VSC (Agile Processes).
- **03-Mathematics** – Minor Subjects (e.g., Operations Research).
- **04-Labs** – Lab course resources.
- **05-Projects** – Mini and Research Projects.
- **06-Revision** – Study materials.
- **07-Exams** – Exam Slips.
- **08-Resources** – Reference books and syllabus.
- **09-Templates** – Standard formats.
- **10-Canvas** – Visual notes.
- **11-Tracking** – Progress trackers.
- **12-Archive** – Archived work.

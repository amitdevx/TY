## 5. Viva Questions
1. **What is the difference between Matplotlib and Seaborn?**
   - Matplotlib is a low-level visualization library offering high customizability. Seaborn is built on top of Matplotlib, provides a higher-level interface, and focuses on beautiful default themes and statistical plotting.
2. **When would you use a scatter plot?**
   - To observe relationships or correlations between two numerical variables.
3. **How do you display multiple plots in a single figure?**
   - By using the `plt.subplot()` function to create a grid of axes.
4. **What is the `tight_layout()` function used for?**
   - It automatically adjusts subplot parameters so that the subplot(s) fits into the figure area without overlapping labels.
5. **What kind of data is best visualized with a bar chart?**
   - Categorical data compared against numerical values.

# Viva Questions: Basics of Machine Learning

1. **What is supervised vs. unsupervised learning?**
   *In supervised learning, the model is trained on labeled data (input-output pairs). In unsupervised learning, the model is trained on unlabeled data, trying to find hidden structures or patterns on its own (e.g., clustering).*

2. **What does Simple Linear Regression aim to achieve?**
   *It aims to find the best-fitting straight line (regression line) that establishes a linear relationship between an independent variable and a dependent target variable, minimizing the prediction error.*

3. **What are the slope (`m`) and intercept (`c`) in linear regression?**
   *The slope represents the rate of change of the dependent variable with respect to the independent variable. The intercept is the predicted value of the dependent variable when the independent variable is zero.*

4. **What is overfitting in Machine Learning?**
   *Overfitting occurs when a model learns the training data too well, including its noise and outliers, resulting in poor generalization and bad performance on new, unseen data.*

5. **Why do we separate data into training and testing sets?**
   *To evaluate the performance of the model objectively. The model learns from the training set, and its ability to generalize to new data is measured using the testing set.*

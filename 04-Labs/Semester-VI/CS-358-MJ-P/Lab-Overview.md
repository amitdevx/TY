## Lab Objectives

By the end of this lab course, students should be able to:
1. **Environment Setup**: Competently navigate and utilize the Android Studio IDE, Android SDK, and Virtual Devices.
2. **UI Design**: Architect and implement responsive user interfaces using XML layouts, ViewGroups, and fundamental Widgets following Material Design principles.
3. **Component Integration**: Understand and orchestrate the core Android components, specifically mastering the Activity Lifecycle and Intent-based navigation.
4. **Data Persistence**: Implement local data storage mechanisms including `SharedPreferences` for user settings and `SQLite` for relational data management.
5. **Advanced Implementation**: Utilize complex UI components like `RecyclerView`, handle background tasks, and perform basic network operations.

---

## Environment Requirements
- **IDE**: Android Studio (Latest Stable Version)
- **Language**: Java / Kotlin
- **Minimum SDK**: API 24 (Android 7.0 Nougat) recommended for broad compatibility.
- **Tools**: Gradle build system, Android Emulator or Physical Device with USB Debugging enabled.

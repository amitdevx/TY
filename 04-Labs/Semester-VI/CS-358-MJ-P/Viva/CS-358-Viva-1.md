## 5. Viva Questions for this Assignment
**Q1. What is the use of `findViewById()`?**
*Answer\*: It links the UI components defined in the XML layout file to the corresponding object variables in the Java/Kotlin code by their unique `android:id`.

**Q2. What does `android:orientation="vertical"` do in a LinearLayout?**
*Answer\*: It stacks all the child views inside the layout one below the other in a single column.

**Q3. How do you extract text from an `EditText` component?**
*Answer\*: Using the method `getText().toString()`.

**Q4. What is a `Toast`?**
*Answer\*: A Toast is a small, non-modal message that briefly appears at the bottom of the screen to provide feedback to the user, and automatically disappears after a timeout.

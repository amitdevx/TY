# CS-355-MJ-P: Design Framework Lab (Angular)

Welcome to the Design Framework Lab. This practical course complements the theoretical concepts taught in **CS-352 Design Framework (Angular)**. It focuses on building robust single-page applications (SPAs) using the Angular framework and TypeScript.

## Core Objectives
1. Understand the architecture of Angular applications (Components, Modules, Services).
2. Implement various types of data binding and structural/attribute directives.
3. Handle user inputs and validations using Template-driven and Reactive forms.
4. Manage application state and business logic using Services and Dependency Injection.
5. Perform asynchronous operations using RxJS Observables and HttpClient.
6. Implement client-side routing and navigation with Route Guards.

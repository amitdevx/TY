## 5. Viva Questions
1. **What is the difference between Interpolation and Property Binding?**
   Interpolation (`{{}}`) converts the value to a string and is best for text nodes. Property binding (`[]`) binds directly to the DOM object property and preserves data types (like booleans for `[disabled]`).
2. **Which module must be imported to use `[(ngModel)]`?**
   `FormsModule` from `@angular/forms`.
3. **What is the syntax for Two-Way Data Binding often called?**
   "Banana in a box" syntax: `[()]`.

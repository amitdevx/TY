# CS-301 Core Java Interview Preparation

## OOP Concepts
1. What is Object-Oriented Programming?
2. Explain the four main concepts of OOP.
3. What is the difference between an Abstract Class and an Interface?
4. How does Java achieve polymorphism?

## Exception Handling
1. What is the difference between Checked and Unchecked Exceptions?
2. Can we write a try block without a catch block?
3. What is the `finally` block used for?
4. What is the difference between `throw` and `throws`?

## Java Fundamentals
1. Why is Java platform independent?
2. What are the differences between JVM, JRE, and JDK?
3. How are Strings immutable in Java?
4. What is a Wrapper class?

## Coding Problems
1. Write a program to reverse a String without using built-in methods.
2. How do you implement a Singleton class?
3. Write a program to check if a number is prime.

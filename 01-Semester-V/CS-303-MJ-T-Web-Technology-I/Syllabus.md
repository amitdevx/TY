# CS-303 Web Technology-I - Complete Syllabus

> [!info] Course Information
> **Semester:** V | **Credits:** 2 | **IE:** 15 | **EE:** 35
> **Teaching:** 2 Hours/Week | **Total:** 30 Hours

## Prerequisites
- Knowledge of Programming Fundamentals
- Basic knowledge of Computer Fundamentals
- Familiarity with basic web browser and web applications

## Course Objectives
- Introduction to scripting languages
- Development of interactive and dynamic client-side applications
- Analyze asynchronous programming concepts for backend development
- Design and implement basic server-side applications and file handling

## Course Outcomes (COs)

| CO | Description |
|----|-------------|
| CO1 | Understand HTML5 elements, CSS3 styling, core JavaScript for frontend |
| CO2 | Develop responsive and interactive web pages using HTML5, CSS3, ES6+ |
| CO3 | Analyze asynchronous programming and Node.js execution model |
| CO4 | Design and implement basic backend apps with HTTP server and file handling |

## Chapter-wise Syllabus

### Chapter 1: Frontend Fundamentals *(9 Hours)*
→ [[Unit-1|Unit 1 Notes]]

#### 1.1 HTML5 Fundamentals
- Structure: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
- Semantic elements: `<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`, `<aside>`
- Forms and input controls: Input types, labels, validation attributes, GET/POST
- Multimedia: Audio, Video, iframe embedding, Accessibility (alt text, captions)

#### 1.2 CSS3 Styling & Layout
- Selectors, specificity, pseudo-classes, pseudo-elements
- **Box model**: margin, padding, border, overflow
- Layout: **Flexbox** (alignment, distribution), **CSS Grid** (rows, columns, areas)
- **Responsive Design**: Media queries, mobile-first approach, fluid images and typography

#### 1.3 Core JavaScript
- Variables, data types, operators
- Conditional statements and loops
- Functions and scope
- Arrays and objects manipulation
- **DOM**: selection, traversal, modification
- Event handling: click, submit, keyboard, load

### Chapter 2: Advanced JavaScript (ES6+) *(5 Hours)*
→ [[Unit-2|Unit 2 Notes]]

| Section | Topics |
|---------|--------|
| 2.1 | **ES6 Syntax**: Arrow functions, lexical `this`, Template literals, Destructuring, Default parameters |
| 2.2 | **Advanced Operators & Modules**: Spread/Rest operators, ES6 modules (import/export) |
| 2.3 | **Async JavaScript**: Callbacks (concept & limitations), Promises (lifecycle: pending/fulfilled/rejected, chaining), **async/await** with try-catch |
| 2.4 | **Logic-Building**: Prime, factorial, Fibonacci; Pattern generation; Searching and sorting |

### Chapter 3: Node.js Fundamentals *(10 Hours)*
→ [[Unit-3|Unit 3 Notes]]

| Section | Topics |
|---------|--------|
| 3.1 | **Node.js Environment**: Runtime architecture, npm, package.json, package management |
| 3.2 | **Execution Model**: Event loop phases, Non-blocking I/O and callbacks |
| 3.3 | **Core Modules**: `fs` (file system), `path` (path utilities), `http` (HTTP server) |
| 3.4 | **Basic Server Development**: Creating and running HTTP server, Handling requests/responses |

> [!tip] Key Concepts
> Node.js is built on Chrome V8 JavaScript engine. Its **event loop** allows non-blocking, asynchronous operations - this is the core concept!

### Chapter 4: File Handling with Node.js *(6 Hours)*
→ [[Unit-4|Unit 4 Notes]]

| Section | Topics |
|---------|--------|
| 4.1 | **File Operations**: Reading, writing, appending, deleting files; Sync vs Async methods |
| 4.2 | **JSON Handling**: Parsing JSON data, writing structured JSON files |
| 4.3 | **Directory Management**: Creating, listing, renaming, deleting folders |
| 4.4 | **File Upload Basics**: Upload middleware intro, handling uploaded file storage |

## Most Important Topics for Exam

> [!warning] Key Topics to Master
> 1. **Flexbox vs Grid** - layout differences and use cases
> 2. **Promise vs async/await** - asynchronous JavaScript
> 3. **Event loop** in Node.js - how it works
> 4. **DOM manipulation** methods
> 5. **ES6 features** - arrow functions, destructuring, spread/rest
> 6. **HTTP server creation** with Node.js
> 7. **File operations** (sync vs async) using `fs` module

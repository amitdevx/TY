# Templates Dashboard

> [!summary] Reusable Note Structures
> Maintain consistency across the vault by duplicating these templates for any new notes or assignments.

## Available Templates
- **[[09-Templates/Unit-Template|Unit Template]]:** Use for creating notes for a new syllabus unit.
- **[[09-Templates/Lecture-Template|Lecture Template]]:** Use for daily classroom notes.
- **[[09-Templates/Lab-Template|Lab Template]]:** Use for practical assignments.
- **[[09-Templates/Weekly-Review-Template|Weekly Review Template]]:** Use for Sunday study reflections.
- **[[09-Templates/Exam-Prep-Template|Exam Prep Template]]:** Standard structure for gathering PYQs and important topics.
- **[[09-Templates/Revision-Template|Revision Template]]:** Use for summarizing chapters before exams.
- **[[09-Templates/Project-Template|Project Template]]:** Use for SRS and project planning.

## How to use Templates
In Obsidian, set the Template folder to `09-Templates`. Use the `Insert Template` command (or `Alt+T` / `Cmd+T`) when creating a new file to quickly inject these structures.

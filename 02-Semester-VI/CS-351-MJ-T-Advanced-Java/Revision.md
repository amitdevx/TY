# CS-351 Advanced Java Quick Revision

- **Collections**: List (ArrayList, LinkedList), Set (HashSet, TreeSet), Map (HashMap, TreeMap). Use `Comparable` (natural ordering) or `Comparator` (custom ordering).
- **Multithreading**: `extends Thread` or `implements Runnable`. Lifecycle: New -> Runnable -> Running -> Waiting/Blocked -> Terminated. `synchronized` prevents race conditions.
- **JDBC**: Driver Types (Type 1-4). Connection, Statement, PreparedStatement, ResultSet. Transactions (commit, rollback).
- **Servlets**: Lifecycle: init(), service(), destroy(). Session Tracking: Cookies, URL rewriting, HttpSession.
- **Spring/Hibernate**: Hibernate maps Java objects to DB tables (ORM). Spring uses Inversion of Control (IoC) and Dependency Injection (DI) to manage object lifecycles.

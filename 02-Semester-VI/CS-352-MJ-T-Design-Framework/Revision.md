# CS-352 Angular Revision

- **Component**: Basic building block, consists of HTML, CSS, and TS class.
- **Directives**: Structural (`*ngIf`, `*ngFor`), Attribute (`ngClass`, `ngStyle`).
- **Data Binding**: Interpolation `{{}}`, Property `[]`, Event `()`, Two-way `[()]`.
- **Routing**: `RouterModule`, `<router-outlet>`, `routerLink`.
- **Guards**: `CanActivate` (protect routes).
- **Services**: `@Injectable()`, used for data sharing and API calls.
- **Forms**: Template-driven (simple, ngModel) vs Reactive (complex, FormGroup).

# CS-352 Important Exam Questions

## Short Answer Questions (2 Marks)
1. What is an SPA?
2. What is Data Binding in Angular?
3. List the component lifecycle hooks.
4. What is the use of `ngOnInit`?
5. Difference between `*ngIf` and `[hidden]`.

1. Explain Angular Architecture with a neat diagram.
2. Differentiate between Template-driven and Reactive forms.
3. Explain Dependency Injection in Angular.
4. How do you implement routing and route guards?
5. Explain the different types of directives in Angular with examples.

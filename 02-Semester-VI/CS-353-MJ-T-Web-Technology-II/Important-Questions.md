# CS-353 Important Exam Questions

## Short Answer Questions (2 Marks)
1. What is the Virtual DOM in React?
2. What are React Hooks?
3. What is JWT and its structure?
4. Explain the purpose of `bcrypt`.
5. What is a RESTful API?

## Long Answer Questions (5 Marks)
1. Explain the MVC architecture in Express.js.
2. Differentiate between Props and State in React.
3. Describe the lifecycle of a React component (Hooks equivalent).
4. Explain session management using express-session.
5. How is password hashing and salting implemented?

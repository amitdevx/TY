# CS-353 Quick Revision

- **PostgreSQL**: Relational DB, connected via `pg` module.
- **REST API**: GET, POST, PUT, DELETE. Resource-based URLs.
- **React**: Component-based UI library. Virtual DOM for performance.
- **Hooks**: `useState` for state management, `useEffect` for side effects.
- **Sessions vs Cookies**: Sessions stored on server, cookies on client.
- **JWT**: Header.Payload.Signature. Used for stateless auth.
- **bcrypt**: Hashing passwords with salt.

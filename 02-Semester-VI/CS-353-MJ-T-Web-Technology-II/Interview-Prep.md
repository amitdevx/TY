# Web Tech II Interview Preparation

1. **Explain the Virtual DOM.**
   A lightweight copy of the actual DOM. React compares the VDOM with a snapshot (diffing) and updates only the changed parts in the real DOM.
2. **What are the advantages of React?**
   Component reusability, fast rendering (VDOM), strong community.
3. **Difference between state and props?**
   Props are immutable and passed down from parent to child. State is mutable and managed within the component.
4. **How do you secure a REST API?**
   Use HTTPS, JWT or session auth, rate limiting, validate inputs.
5. **What is CORS?**
   Cross-Origin Resource Sharing. A mechanism that allows restricted resources to be requested from another domain.

# CS-354 Quick Revision

- **Lexical Analysis**: FA, Tokens, Lexemes. Removes whitespace/comments.
- **Syntax Analysis**: Parse trees, LL(1), LR(1), SLR, CLR, LALR.
- **Semantic Analysis**: Type checking, symbol table.
- **IR Generation**: 3AC, Quadruples, Triples.
- **Optimization**: Constant folding, Dead code elimination, Loop invariant movement.
- **Code Generation**: Register allocation, target code.
- **LL(1) parsing**: Requires elimination of Left Recursion and Left Factoring.
- **LR Parsers**: CLR is the most powerful. LALR is widely used (e.g., YACC).

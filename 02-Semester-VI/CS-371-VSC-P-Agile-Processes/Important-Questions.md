# CS-371 Important Exam Questions

## Short Answer Questions (2 Marks)
1. List the 4 values of the Agile Manifesto.
2. What is a User Story?
3. What is Definition of Done (DoD)?
4. Explain Story Points.
5. What is the role of a Scrum Master?

## Long Answer Questions (5 Marks)
1. Explain the Agile Manifesto and its 12 principles.
2. Describe the Scrum Framework (Roles, Events, Artifacts).
3. How is Sprint Planning conducted?
4. Explain MoSCoW prioritization technique.
5. What is a Burndown chart and how is it useful?

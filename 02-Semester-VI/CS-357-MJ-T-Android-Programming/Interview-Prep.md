# Android Interview Preparation

1. **What is an Intent?**
   A messaging object used to request an action from another app component.
2. **Difference between Activity and Fragment?**
   An Activity is an independent screen. A Fragment is a modular section of an Activity, with its own lifecycle.
3. **What is RecyclerView and why is it better than ListView?**
   RecyclerView is more advanced and flexible. It reuses views (view holder pattern) for better performance.
4. **Explain MVVM.**
   Model provides data. ViewModel holds UI logic and data state. View displays the data. It decouples UI from business logic.
5. **How does Room database work?**
   It's an abstraction layer over SQLite. Uses @Entity (table), @Dao (queries), and @Database.

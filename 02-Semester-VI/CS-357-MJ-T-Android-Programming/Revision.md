# CS-357 Quick Revision

- **Architecture**: Linux Kernel, HAL, ART, Framework, App layer.
- **Lifecycle**: onCreate -> onStart -> onResume -> onPause -> onStop -> onDestroy.
- **Intents**: Explicit (start specific component), Implicit (action to be performed).
- **Layouts**: LinearLayout, RelativeLayout, ConstraintLayout.
- **Storage**: SharedPreferences (key-value), SQLite (relational), Room (ORM).
- **Networking**: Retrofit (REST client), JSON parsing (Gson).
- **Advanced**: MVVM (Model-View-ViewModel), LiveData (observable), Services (background tasks).

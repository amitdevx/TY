## Exam Strategies

### General Time Management for a 2-Hour UE Paper

> [!tip]
> Allocate time proportionally to marks. For a 35-mark paper in 2 hours:
> - Section A (10 marks): 25-30 minutes (3 minutes per 2-mark question)
> - Section B (20 marks): 60-70 minutes (15-18 minutes per 5-mark question)
> - Section C (5 marks): 20-25 minutes
> - Buffer / Review: 5-10 minutes

### Answering Strategy by Question Type

| Question Type | Strategy |
|---|---|
| 2-mark (Define / State) | Give formal definition + one key property or example. Aim for 3-4 lines. |
| 5-mark (Explain / Describe) | Introduction (1-2 lines) + diagram or table + explanation + example. Use headings. |
| 5-mark (Design / Solve) | Show working step by step. For algorithms, trace with a small example. Write conclusion. |
| 10-mark (not in current pattern) | Divide into sub-sections. Use numbered points, tables, diagrams. |

### Subject-Specific Exam Tips

> [!tip]
> **Core Java (CS-301):**
> Write syntactically correct Java code with proper indentation. For OOP questions, always explain the concept before showing code. Diagrams for class hierarchies and JavaFX scene graph are worth drawing.

> [!tip]
> **Operating Systems (CS-302):**
> For scheduling problems, always draw the Gantt chart and show the calculation table for TAT and WT. For Banker's Algorithm, show all data structures (Max, Allocation, Need, Available) and trace the safety algorithm step by step.

> [!tip]
> **Theory of Computer Science (CS-304):**
> For DFA/NFA questions, draw state diagrams and include the transition table. For Pumping Lemma proofs, state the lemma formally before applying it. For grammar problems, show the derivation steps clearly.

## Quick Reference - Key Formulas and Facts

### OS Scheduling Formulas

$$\text{Turnaround Time (TAT)} = \text{Completion Time} - \text{Arrival Time}$$
$$\text{Waiting Time (WT)} = \text{TAT} - \text{Burst Time}$$
$$\text{Response Time} = \text{First CPU Allocation} - \text{Arrival Time}$$
$$\text{Effective Access Time (EAT)} = h \times (t_{TLB} + t_{mem}) + (1-h) \times (t_{TLB} + 2 \times t_{mem})$$

### TCS Key Facts

| Fact | Statement |
|---|---|
| Regular $\subset$ CFL | $\{a^nb^n\}$ is CFL but not regular |
| CFL $\subset$ CSL | $\{a^nb^nc^n\}$ is CSL but not CFL |
| CFL closure | Closed under union, concatenation, Kleene star; NOT under intersection or complement |
| Regular closure | Closed under all regular operations AND complement AND intersection |
| Pumping length | For regular: 3-part pump; for CFL: 5-part pump |
